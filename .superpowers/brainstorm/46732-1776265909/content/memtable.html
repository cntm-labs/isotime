LSM-Tree Component: MemTable Selection

MemTable คือด่านแรกที่รับข้อมูล ก่อนจะถูก Flush ลง Disk เป็น SSTable

A

Lock-free SkipList

มาตรฐานสำหรับระบบที่ต้องการ High Concurrency (เหมือน RocksDB) รองรับการเขียนพร้อมกันได้ดีมาก

B

Concurrent B-Tree

เน้นประสิทธิภาพด้าน Cache Locality และการค้นหาที่รวดเร็ว (เหมือน Sled) เหมาะกับสถาปัตยกรรม CPU สมัยใหม่

C

Fixed-size Ring Buffer (Specialized)

ออกแบบมาเพื่อ Time-series Delta โดยเฉพาะ เน้นการเขียนที่เร็วที่สุดและจัดการ Memory ได้แม่นยำ (Predictable)
