# Project Governance

## 🏛️ Decision Making
This project follows a meritocratic governance model. Core contributors have the authority to make technical decisions.

## 👥 Roles
- **Maintainers:** Lead the project direction and manage releases.
- **Contributors:** Provide code, documentation, and feedback.
- **Users:** Utilize the project and report issues.

## ⚖️ Transparency
All significant changes are discussed in Public Issues and Pull Requests.
