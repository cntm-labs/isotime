# Support & Help

## 🆘 Getting Help
If you run into issues with `isotime`, here is how you can get help:

1. **Check the Docs:** Start with the [README.md](README.md) and [ARCHITECTURE.md](ARCHITECTURE.md).
2. **Search Issues:** See if someone else has already reported the problem.
3. **Open a Bug Report:** Use the [template](.github/ISSUE_TEMPLATE/bug_report.md).

## 🛡️ Security Issues
Do NOT open public issues for security vulnerabilities. Please follow the [SECURITY.md](SECURITY.md) policy.
