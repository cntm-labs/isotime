จุดเด่นของการออกแบบนี้

  • Write Speed: เขียนลง WAL และ RAM ได้ทันที รองรับ High-throughput
  • Reliability: ข้อมูลไม่หายด้วย WAL
  • Query Efficiency: กรองข้อมูลตามช่วงเวลาได้รวดเร็ว (Time Pruning)
  • Maintainability: การจัดการไฟล์แยกตามเวลาทำให้ Debug และลบข้อมูลเก่าได้ง่าย
