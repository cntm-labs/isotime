# Frequently Asked Questions (FAQ)

## ❓ General
**Q: What is isotime?**
A: High-Throughput Time-Series Engine for Graph Deltas

## 🛠️ Technical
**Q: How do I run tests?**
A: Use the command: `cargo test`.

**Q: Which languages are supported?**
A: Primarily Rust (Tokio/io_uring), Custom LSM-Tree, FlatBuffers (Zero-copy), SIMD (AVX-512).
